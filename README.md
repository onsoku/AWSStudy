# AWS Study App

AWS認定資格（SAA、CloudOps、Security Specialty、SAP）の取得を目指す学習支援アプリケーションです。
AIを活用した動的な問題生成・ディスカッションにより、実践的な知識の習得をサポートします。

## 機能
- **クイズ自動生成:** ユーザーの現在レベルと選択資格のトピックに合わせて、AI（Claude）がリアルタイムにJSONで問題を生成します。
- **直感的な正誤判定機能:** 回答後、すぐに正解ハイライトと詳細な解説が表示され、成績が記録されます。
- **AIディスカッション:** クイズで間違えた問題やわからない用語を、継続的なコンテキストを持つAIコーチに直接質問できます。
- **統計・レベル進行:** 回答履歴（ローカルストレージに保存）に基づき、各資格ごとにレベル（Lv1〜5）を動的に算出し、全体の習熟度をグラフで可視化します。

## 開発・起動方法

本プロジェクトは React + Vite 環境で構成されています。（※本番環境として、`App.jsx` 単体を抽出して Claude Artifacts 環境でそのまま動作させることも考慮された設計になっています。）

### インストールと起動
```bash
npm install
npm run dev
```

### APIキーの設定
ブラウザのローカル実行環境からAIクイズ生成・チャット機能をご利用になる場合は、画面上の「HELP / SETTINGS」タブから [Anthropic API Key](https://console.anthropic.com/) を設定してください。（APIキーはブラウザの `localStorage` に安全に保存され、サーバー等には送信されません。）
※ キーが設定されていない場合は、UI検証用として問題やチャットのモック（ダミーデータ）が高速で返却されます。
